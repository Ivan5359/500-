Railway:

1. Если создаёшь новый репозиторий, загрузи содержимое этой папки в корень репозитория.
   Тогда Railway сам увидит package.json и запустит npm start.

2. Если загружаешь эту папку внутрь старого репозитория, в Railway поставь:
   Root Directory: RAILWAY_READY_REVENUESPRINT

3. Start Command:
   npm start

4. Healthcheck:
   /health

После деплоя открывай:

https://твоя-ссылка.up.railway.app/?v=clean-20260616
